# Roadmap

## Near term

- Add MTProto-based channel statistics ingestion (reach, views dynamics, engagement quality)
- Build normalized analytics snapshots from MTProto signals for deal scoring
- Add stricter validation for scheduling windows
- Extend notifications for every critical transition

## Mid term

- Add repeat deal flows (one click relaunch for successful placements)
- Add reusable deal templates for recurring advertiser campaigns
- Add multi creative support in one deal
- Add stronger payout batching and fee optimization

## Long term

- Add AI matching for advertisers (best channels by budget, niche, conversion assumptions)
- Add AI recommendations for optimal posting windows and creative format fit
- Add reputation scoring for advertisers and publishers
- Add deeper fraud detection based on posting behavior
