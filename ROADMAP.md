- Add richer deal timeline API for mini app UI
- Improve analytics snapshots for channel quality signals
- Add stricter validation for scheduling windows
- Extend notifications for every critical transition

- Add multi creative support in one deal

- Add reputation scoring for advertisers and publishers
- Add campaign bundles and recurring placements
- Add deeper fraud detection based on posting behavior
- Add cross chain payment options while keeping escrow first model
